HTML Basic Tags

This lesson will teach you the basic html elements

The header tag

The header element represents a container for introductory content or a set of navigational links.
A header element typically contains: one or more heading elements (h1 - h6) logo or icon
authorship information HTML elements have opening tags like h1 and closing tags like h1

The paragraph tag

The 'p' tag defines a paragraph.
Browsers automatically add a single blank line before and after each 'p' element.

The image tag

The 'img' tag defines a placeholder to add images.


The alt tag

The 'alt' tag will show text on an image if the image is not available.


The anchor tag

The anchor tag will allow an image or text to be clicked and redirect the user to a link.
The item you wanted to be converted to a link should be inside the anchor tag.


The target attribute


The target attribute will allow an anchor to move into the value of the target element.
